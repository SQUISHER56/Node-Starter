in modo take da colegare due diversi file JavaScript, esistono due 
modi per fare questo:
------------------------------------------------------
1)
    nominare il file con l'indice .mjs

    ------------------------------------

    ES:

    file.mjs

    ------------------------------------

-------------------------------------------------------
2)
    all'interno del file packege-json inserire il "type":"module"
    e nominare i file con l'indice .js
